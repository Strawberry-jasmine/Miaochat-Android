# Miaochat for Android v1.0.0

## 这版有什么

- 聊天页现在带有 thread history rail，可以直接查看、切换、搜索和管理最近会话。
- 回复可以在应用退到后台后继续运行，避免长回复因为切到别的应用就中断。
- 设置页整理成了更完整的发布版入口，包含语言、主题、快速预设、实时配置摘要和导入配置。
- 聊天过程里的状态提示更清楚了，能区分 thinking、searching 和 streaming。
- `intelalloc` 预设的基础地址已经改成 `https://backend.intelalloc.com`。

## 发布信息

- 应用名：`Miaochat`
- 包名：`com.example.relaychat`
